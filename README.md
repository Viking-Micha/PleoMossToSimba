# PleoMossToSimba

Konvertiert Pleo- und MOSS-Exportdateien in DATEV/SIMBA-kompatible Importpakete.

## Funktionen

- Konvertierung von Pleo-Exporten
- Konvertierung von MOSS-Exporten
- Erzeugung einer DATEV-/EXTF-kompatiblen ZIP-Datei
- Übernahme von Belegen aus ZIP-Exporten
- Aufbereitung für den Import in SIMBA

## Hinweis

Das Tool wurde für den praktischen Einsatz beim Import von Exportdaten nach DATEV/SIMBA gebaut.  
Nutzung auf eigene Gefahr – wie immer, wenn CSVs und Buchhaltungssoftware sich zum Tanz treffen.
